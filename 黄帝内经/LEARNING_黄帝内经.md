## 基础理论（Week 1-2）

### 一、阴阳学说

**基本概念**：
| 阳 | 阴 |
|---|---|
| 天 | 地 |
| 日 | 月 |
| 昼 | 夜 |
| 热 | 寒 |
| 动 | 静 |
| 表 | 里 |
| 腑 | 脏 |
| 气 | 血 |

**阴阳关系**：
```
互根：孤阴不生，独阳不长
消长：阴长阳消，阳长阴消
转化：重阴必阳，重阳必阴
平衡：阴平阳秘，精神乃治
```

**经典原文**：
> "阴阳者，天地之道也，万物之纲纪，变化之父母，生杀之本始，神明之府也。"
> — 《素问·阴阳应象大论》

### 二、五行学说

**五行属性**：
| 木 | 火 | 土 | 金 | 水 |
|---|---|---|---|---|
| 肝 | 心 | 脾 | 肺 | 肾 |
| 胆 | 小肠 | 胃 | 大肠 | 膀胱 |
| 春 | 夏 | 长夏 | 秋 | 冬 |
| 生 | 长 | 化 | 收 | 藏 |
| 青 | 赤 | 黄 | 白 | 黑 |
| 酸 | 苦 | 甘 | 辛 | 咸 |
| 怒 | 喜 | 思 | 悲 | 恐 |
| 目 | 舌 | 口 | 鼻 | 耳 |

**生克关系**：
```
相生（木生火→火生土→土生金→金生水→水生木）
相克（木克土→土克水→水克火→火克金→金克木）
相乘（克得太过）
相侮（反克）
```

## 素问精读（Week 3-9）

### 上古天真论（Week 3）— 养生总纲

**核心观点**：
```
上古之人：
- 法于阴阳
- 和于术数
- 食饮有节
- 起居有常
- 不妄作劳

→ 形与神俱，尽终天年

今时之人：
- 以酒为浆
- 以妄为常
- 醉以入房
- 不知持满
- 逆于生乐

→ 半百而衰
```

**养生四种境界**：
| 境界 | 特征 | 寿命 |
|-----|------|------|
| 真人 | 提挈天地，把握阴阳 | 度百岁乃去 |
| 至人 | 淳德全道，和于阴阳 | 亦可百数 |
| 圣人 | 处天地之和，从八风之理 | 形骸不敝，精神不散，亦百数 |
| 贤人 | 法则天地，象似日月 | 可使益寿而有极时 |

### 生气通天论（Week 4）— 阴阳与健康

**核心**：
```
阳气者，若天与日
失其所，则折寿而不彰

阳气的作用：
- 卫外而为固
- 滋养五脏
- 维持体温
```

**阳气损伤的原因**：
```
- 寒邪入体
- 暑邪伤气
- 湿邪困脾
- 风邪袭表
- 烦劳过度
- 大怒伤阳
- 饮食不节
```

**作息**：
```
早起（卯时5-7点）→ 排便
早餐（辰时7-9点）→ 养胃
午休（午时11-13点）→ 养心
晚餐（酉时17-19点）→ 养肾

```

夜宵要戒掉

五味调和，不偏不嗜
