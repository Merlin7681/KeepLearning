# 第三篇：生气通天论

## 基本信息
- **篇名**：生气通天论
- **出处**：《黄帝内经·素问》第三篇
- **主题**：人的生气与天相通，万病皆因气机失调

---

## 一、开篇第一句话：本篇总纲

> **"夫自古通天者，生之本，本于阴阳。"**

**逐字拆解**：
- **通天**：人的生命之气与天地之气相连相通
- **生之本**：生命存在的根本依据
- **本于阴阳**：根本在于阴阳的动态平衡

**现代理解**：人不是孤立的系统，是与自然环境连成一个巨大的能量网络。这个网络的核心是阴阳两股力量的平衡。

---

## 二、人体的通天网络

> **"天地之间，六合之内，其气九州、九窍、五脏、十二节，皆通乎天气。"**

| 人体部位 | 具体构成 | 对应说明 |
|---------|---------|---------|
| 九州 | 古代地理概念 | 借指人体各个区域 |
| 九窍 | 眼2、耳2、鼻2、口1、二阴2 | 与外界直接连通 |
| 五脏 | 心肝脾肺肾 | 内脏系统 |
| 十二节 | 四肢八大关节 + 左右肩胛 + 左右髋 | 全身经脉汇聚处 |

**核心含义**：从头到脚，从五官到内脏到关节，全部与"天气"相连。人就是一个小宇宙。

---

## 三、阴阳：万物的总规律

> **"阴阳者，天地之道也，万物之纲纪，变化之父母，生杀之本始，神明之府也。治病必求于本。"**

这是《内经》中对阴阳最完整的定义，连续用了五个比喻：

| 比喻 | 含义 |
|------|------|
| **天地之道** | 阴阳是天地运行的最高法则 |
| **万物之纲纪** | 阴阳是统领万物运作的总纲 |
| **变化之父母** | 一切变化都从阴阳中来 |
| **生杀之本始** | 生长和衰亡的根本原因 |
| **神明之府** | 神明（生命活动）的所在之处 |

→ **"治病必求于本"** — 这个"本"就是阴阳。

**临床含义**：不管什么病，都要在阴阳失衡上找原因。高血压是阳亢，阴虚火旺是阴亏。调理的不是症状，是背后的阴阳格局。

---

## 四、阳气：生命之火

本篇用大量篇幅讲阳气，因为**阳气是人体抵御外邪、维持生命活动的第一道屏障**。

### 阳气的作用
> **"阳气者，若天与日，失其所则折寿而不彰。"**

这句话把阳气比作天上的太阳——人没有阳气，就像天没有太阳，万物无法生长。

### 阳气的四大功能

| 功能 | 说明 |
|------|------|
| **温煦** | 温暖身体，维持体温 |
| **卫外** | 保卫体表，抵御外邪（风寒暑湿燥火） |
| **固摄** | 固护津液，防止汗液妄泄 |
| **气化** | 推动气血运行，维持脏腑功能 |

### 阳气受损的三种典型情形

**① 过度烦劳 → 煎厥**
> "因而劳役，阳气者亢而害筋肉，而为煎厥。"

现代解读：长期过度劳累、熬夜，阳气亢盛而耗伤阴精，形成"煎厥"——类似现代医学的热射病或极度疲劳综合征。

→ **启示**：牛市中高杠杆持仓，就像"过度烦劳"，阳气亢盛，一遇市场暴跌（外邪），直接"煎厥"爆仓。

**② 感受风邪 → 偏枯**
> "风者，百病之始也。入阳则避，入阴则伏。"

风邪是百病的源头，是最先攻击人体表层的病邪。

五脏分论（风邪入侵的层次）：

| 层次 | 症状 | 病机 |
|------|------|------|
| 在表 | 汗出、身重、倦怠 | 阳气受阻，津液不固 |
| 在络 | 四肢酸痛、麻木 | 风邪阻络 |
| 传经 | 寒热往来 | 病邪深入脏腑 |
| 入脏 | 神志昏乱、生命危殆 | 病邪深入阴分 |

**③ 大怒伤阳 → 薄厥**
> "阳气者，大怒则气绝，血郁于上，使人薄厥。"

大怒之下阳气逆乱，血随气上冲头部，导致"薄厥"（突发昏厥）。这说明**情绪剧烈波动是伤阳最快的途径之一**。

→ **启示**：投资中大亏后情绪崩溃（愤怒、恐惧），会进一步做出错误决策，形成恶性循环。

## 五、阴精：阳气的根基

> **"阴者，藏精而起亟也；阳者，卫外而为固也。"**

阴的作用是**藏精**——把精气储存起来，以备不时之需。
阳的作用是**卫外**——保护体表，抵御外邪。

**二者关系**：
```
阴（藏精） → 支持阳气持续输出
阳（卫外） → 保护阴精不外泄
```

→ **阴在内，阳之守也；阳在外，阴之使也。**

**投资对照**：
- **阴精** = 本金（资金储备）
- **阳气** = 交易能力（把握机会的能力）
- 有本金才能持续交易（阴藏精，阳卫外）
- 保护好本金才能发挥交易能力（阴为阳之守）

## 六、饮食五味与阴阳调和

> **"阴之所生，本在五味；精之所伤，亦在五味。"**

五味（酸苦甘辛咸）入五脏，用好了是药，用过头是毒。

| 五味 | 入脏 | 补益 | 过量之害 |
|------|------|------|---------|
| 酸 | 肝 | 补肝敛气 | 伤脾（肝木克脾土） |
| 苦 | 心 | 补心清热 | 伤肺（心火刑肺金） |
| 甘 | 脾 | 补脾和中 | 伤肾（脾土克肾水） |
| 辛 | 肺 | 补肺发散 | 伤肝（肺金克肝木） |
| 咸 | 肾 | 补肾软坚 | 伤心（肾水凌心火） |

**核心智慧**：五味调和是补，偏嗜某味是毒。投资也一样——**单一策略（偏嗜一味）长期必伤**。成长股好，但全仓成长就是偏食；价值投资好，但全仓低估也危险。

## 七、最震撼的一句话

> **"圣人传精神，服天气，而通神明。"**

三个层次的通：

1. **通天气** — 顺应四时寒暑，与自然节律同频
2. **通神明** — 精神内守，神清气明，不被外物扰动
3. **通天地** — 人与宇宙合一，天地人三才共振

**如何做到**：
- "苍天之气，清净则志意治" → 心要清静
- "顺之则阳气固，虽有贼邪，弗能害也" → 顺应天道，外邪难侵

## 八、本篇的千古名句

> **"阴平阳秘，精神乃治；阴阳离决，精气乃绝。"**

| 状态 | 结果 |
|------|------|
| 阴平阳秘 | 精神乃治（身体健康，精神安定） |

## 十一、思考与感悟

### 感悟一：天人相应的系统世界观
古人没有现代医学，但，他们用"气"这个概念把人体和宇宙联系在一起。现在看来，这个"气"对应的是现代科学中的**能量代谢**、**神经内分泌**、**免疫系统**——所有维持生命运转的能量和信息流。与其说是玄学，不如说是高度抽象的系统论。

### 感悟二：阳气是核心资产
"阳气者，若天与日，失其所则折寿而不彰"——阳气就像太阳，是生命的第一资源。在投资中，**本金就是阳气**，失去了就什么都没了。所以止损不是失败，是保护阳气；仓位管理不是保守，是养阳气。

### 感悟三：煎厥与杠杆
"烦劳则张，精绝，煎厥"——过度劳累导致阳气亢盛、精气耗竭。这和杠杆炒股的处境一模一样：加了杠杆就像"烦劳"，行情稍有不利就会"精绝"（本金大幅亏损），然后"煎厥"（爆仓离场）。所以巴菲特说"永远不要用杠杆"是真正懂"生气通天"的道理。

### 感悟四：治未病就是防微杜渐
"是故圣人不治已病治未病"——圣人在病还没发生时就预防了，不是在病发生后去治疗。这在投资中就是：**熊市初期控制仓位，减少损失**，而不是等到深套了才想办法解套。
