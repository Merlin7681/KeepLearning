> **需：有孚，光亨，贞吉。利涉大川。**

| 关键词 | 含义 |

| 有孚 | 心中有信——诚信、信心、信念 |
| 光亨 | 光明亨通——前途是光明的 |
| 贞吉 | 守正则吉——但必须守住正道 |

> 乾坤屯蒙之后，万物需要滋养——需卦讲的就是这个"养"字。

## 三、彖传

> "需，须也，险在前也。刚健而不陷，其义不困穷矣。"

险在前方——前方有坎水（危险），但乾体刚健，不会陷进去。

**刚健而不陷** — 这是需卦的精髓：有实力、有信心，所以等得住；等得住，所以不会困穷。

### 🔍 六爻递进：等待的六个阶段

从郊野到洞穴，危险越来越近，应对方式不同：

```
郊野（远） → 沙地（稍近） → 泥泞（太近） → 血泊（身陷） → 酒食（从容） → 洞穴（退守）
```

**关键转折在九五**：前四爻是"被迫等待"，到九五变成"从容等待"——这是质的飞跃。

| 阶段 | 爻 | 心态 | 投资对应 |
|------|-----|------|---------|
| 轻松等待 | 初九·郊 | 淡定，离风险远 | 空仓观望，市场还没到底 |
| 有些议论 | 九二·沙 | 有人催你出手 | 别人说"你怎么还不买"，不为所动 |
| 靠太近了 | 九三·泥 | 心痒手痒 | 差一点就追进去了，忍住 |
| 被套住了 | 六四·血 | 已经入场被套 | 冷静止损/持有等反弹，别加仓 |
| 从容应对 | 九五·酒食 | 等到最好的位置 | 底部从容建仓，不为波动所动 |
| 意外来了 | 上六·穴 | 突发黑天鹅 | 不请自来的危机，冷静应对，终能化解 |

### 🔍 九五爻精读：需于酒食，贞吉

九五是需卦的主心骨——阳居尊位，得中得正。

**"需于酒食"** 不是吃喝享乐，而是：

1. **等待的条件已经成熟**——你有酒有食，有实力、有准备
2. **心态从容**——不焦虑、不慌张，因为知道自己等得起
3. **守正则吉**——但仍然不能冒进，守住了才是真吉

> 这是投资中最好的状态：**研究做透了、资金到位了、仓位控制好了，然后在底部从容地买**。
