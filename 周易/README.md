## 🎯 学习方式

- **每日一卦**，每天讲完自动记录到当日子目录
- 每个卦固定结构：卦象速查 → 卦辞 → 彖传 → 象传 → 六爻详解 → 投资映射 → 核心关键词
- 结合 MEMORY.md 中的仓位体系、东方智慧框架进行投资视角解读
- 每卦笔记存放于：`~/Documents/GitHub/Learning/周易/YYYY-MM-DD/`
