## 基础概念（Week 1-2）

### 必背内容

**八卦口诀**：
```
乾三连 ☰  坤三断 ☷
震仰盂 ☳  艮覆碗 ☶
离中虚 ☲  坎中满 ☵
兑上缺 ☱  巽下断 ☴
```

**八卦类象速查**：
| 卦 | 自然 | 性情 | 方位 | 季节 |
|---|------|------|------|------|
| ☰ 乾 | 天 | 刚健 | 西北 | 秋 |
| ☱ 兑 | 泽 | 喜悦 | 西 | 秋 |
| ☲ 离 | 火 | 明丽 | 南 | 夏 |
| ☳ 震 | 雷 | 震动 | 东 | 春 |
| ☴ 巽 | 风 | 进入 | 东南 | 春 |
| ☵ 坎 | 水 | 险陷 | 北 | 冬 |
| ☶ 艮 | 山 | 静止 | 东北 | 冬 |
| ☷ 坤 | 地 | 柔顺 | 西南 | 夏 |

**三大原则**：
```
变易 — 唯一不变的就是变化
不易 — 变化中有不变的规律
简易 — 复杂中蕴含简单大道
```

## 学习方法

### 每日一卦流程

```
1. 读卦象（符号+卦名+卦辞）
2. 读爻辞（6个爻的变化）
3. 查象传（孔子解释）
4. 思考含义（联系生活）
5. 记录心得
```
