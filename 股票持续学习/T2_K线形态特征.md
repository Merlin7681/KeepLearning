## 一、单根K线形态

### 1. 阳线与阴线

| 形态 | 特征 | 市场含义 |
|-----|------|---------|
| **大阳线** | 实体长，收盘远高于开盘 | 强势多头，买方力量强 |
| **大阴线** | 实体长，收盘远低于开盘 | 强势空头，卖方力量强 |
| **小阳线** | 实体短，波动小 | 多头犹豫，力量弱 |
| **小阴线** | 实体短，波动小 | 空头犹豫，力量弱 |

---

### 2. 关键反转形态

#### 🔴 锤子线（底部反转）
```
  |
  |
 ─┴─
  |
```
- **特征**：实体小（可阳可阴），下影线≥实体2倍，上影线很短或没有
- **位置**：下跌趋势末端
- **含义**：空头打压被多头接回，见底信号
- **确认**：次日收阳线确认

#### 🔵 上吊线（顶部反转）
```
  |
 ─┬─
  |
  |
```
- **特征**：形态同锤子线，但出现在上涨趋势末端
- **含义**：多头力量衰竭，见顶信号
- **确认**：次日收阴线确认

#### 🔴 倒锤子线（底部反转）
```
  |
  |
 ─┬─
```
- **特征**：实体小，上影线≥实体2倍，下影线很短或没有
- **位置**：下跌趋势末端
- **含义**：多头尝试上攻被压制，但空头已无力继续
- **确认**：次日高开或收阳线确认

#### 🔵 流星线（顶部反转）
```
  |
  |
 ─┴─
```
- **特征**：形态同倒锤子线，但出现在上涨趋势末端
- **含义**：多头冲高回落，见顶信号
- **确认**：次日收阴线确认

---

### 3. 十字星系列

#### ⚖️ 普通十字星
```
 ─┼─
```
- **特征**：开盘价=收盘价，实体极小或没有
- **含义**：多空力量平衡，变盘信号
- **位置判断**：
  - 上涨趋势末端 → 见顶信号
  - 下跌趋势末端 → 见底信号
  - 盘整中 → 延续震荡

#### 📍 长腿十字星
```
  |
  |
 ─┼─
  |
  |
```
- **特征**：上下影线都很长
- **含义**：剧烈博弈后回归平衡，强烈的变盘信号

#### 📍 蜻蜓十字星
```
 ─┬─
  |
  |
```
- **特征**：只有下影线，无上影线
- **含义**：多头强势接盘，底部信号

#### 📍 墓碑十字星
```
  |
  |
 ─┴─
```
- **特征**：只有上影线，无下影线
- **含义**：空头强势打压，顶部信号

## 二、双根K线形态

### 1. 吞没形态（抱线）

#### ✅ 看涨吞没（底部反转）
```
 Day1:  ─┴─
 Day2: ─┬──┴─
```
- **特征**：阴线后出现更大的阳线，阳线实体完全包含阴线实体
- **位置**：下跌趋势末端
- **含义**：多头力量压倒空头，强反转信号
- **强度**：⭐⭐⭐⭐

#### ❌ 看跌吞没（顶部反转）
```
 Day1:  ─┬─
 Day2: ─┴──┬─
```
- **特征**：阳线后出现更大的阴线，阴线实体完全包含阳线实体
- **位置**：上涨趋势末端
- **含义**：空头力量压倒多头，强反转信号
- **强度**：⭐⭐⭐⭐

---

### 2. 孕线形态（身怀六甲）

#### ✅ 看涨孕线
```
 Day1: ─┴──┬─
 Day2:   ─┬─
```
- **特征**：大阴线后出现小K线，小K线完全在大阴线实体内
- **位置**：下跌趋势末端
- **含义**：空头力量减弱，可能见底
- **强度**：⭐⭐⭐

#### ❌ 看跌孕线
```
 Day1: ─┬──┴─
 Day2:   ─┴─
```
- **特征**：大阳线后出现小K线，小K线完全在大阳线实体内
- **位置**：上涨趋势末端
- **含义**：多头力量减弱，可能见顶
- **强度**：⭐⭐⭐

---

### 3. 乌云盖顶与刺透形态

#### ❌ 乌云盖顶（顶部反转）
```
    Day1:  ─┬──┴─
    Day2:  ─┴──┬─
          ─────── 前日中点
```
- **特征**：阳线后高开，阴线收盘深入阳线实体内部（超过50%）
- **位置**：上涨趋势末端
- **含义**：多头被空头压制，见顶信号
- **强度**：⭐⭐⭐⭐

#### ✅ 刺透形态（底部反转）
```
    Day2:  ─┬──┴─
    Day1:  ─┴──┬─
          ─────── 前日中点
```
- **特征**：阴线后低开，阳线收盘深入阴线实体内部（超过50%）
- **位置**：下跌趋势末端
- **含义**：空头被多头压制，见底信号
- **强度**：⭐⭐⭐⭐

## 三、三根K线形态

### 1. 三白兵（底部反转）
```
 Day1: ─┬─
 Day2:  ─┬─
 Day3:   ─┬─
```
- **特征**：三根连续阳线，每根开盘在前根实体内，收盘创新高
- **位置**：下跌趋势末端或盘整突破
- **含义**：多头力量持续增强，强反转信号
- **强度**：⭐⭐⭐⭐⭐

### 2. 三只乌鸦（顶部反转）
```
 Day1: ─┴─
 Day2:  ─┴─
 Day3:   ─┴─
```
- **特征**：三根连续阴线，每根开盘在前根实体内，收盘创新低
- **位置**：上涨趋势末端
- **含义**：空头力量持续增强，强反转信号
- **强度**：⭐⭐⭐⭐⭐

### 3. 早晨之星（底部反转）
```
 Day1: ─┴─
 Day2:   ─┼─
 Day3: ─┬──┴─
```
- **特征**：大阴线 + 跳空小K线（十字星最优） + 跳空大阳线
- **位置**：下跌趋势末端
- **含义**：空头衰竭，多头反击，最强反转信号之一
- **强度**：⭐⭐⭐⭐⭐

### 4. 黄昏之星（顶部反转）
```
 Day1: ─┬─
 Day2:   ─┼─
 Day3: ─┴──┬─
```
- **特征**：大阳线 + 跳空小K线（十字星最优） + 跳空大阴线
- **位置**：上涨趋势末端
- **含义**：多头衰竭，空头反击，最强反转信号之一
- **强度**：⭐⭐⭐⭐⭐

## 四、多根K线形态

### 1. 头肩顶
```
     ●左肩  ●头  ●右肩
    ─┬─    ─┬─
   ─┴─    ─┴─
  ─┴─    ─┴─    ─┴─
        颈线 ──────────
```
- **特征**：左肩（高点）→ 头（最高点）→ 右肩（次高点），两肩高度相近
- **颈线**：连接两次回调低点
- **确认**：跌破颈线，跌幅至少=头顶到颈线距离
- **含义**：大级别顶部，看跌
- **强度**：⭐⭐⭐⭐⭐

### 2. 头肩底
```
        颈线 ──────────
  ─┬─    ─┬─    ─┬─
   ─┬─    ─┬─
    ─┴─  ─┴─  ─┴─
     ●左肩 ●头  ●右肩
```
- **特征**：倒置的头肩顶形态
- **确认**：突破颈线，涨幅至少=头顶到颈线距离
- **含义**：大级别底部，看涨
- **强度**：⭐⭐⭐⭐⭐

---

### 3. 双底（W底）
```
     ─┬─
    ─┴─    ─┬─
   ─┴─    ─┴─
       颈线 ────
```
- **特征**：两个低点相近（第二个可略高），中间有反弹
- **确认**：突破颈线
- **含义**：底部反转
- **强度**：⭐⭐⭐⭐

### 4. 双顶（M顶）
```
       颈线 ────
    ─┬─    ─┬─
   ─┴─    ─┴─
     ─┴─
```
- **特征**：两个高点相近（第二个可略低），中间有回调
- **确认**：跌破颈线
- **含义**：顶部反转
- **强度**：⭐⭐⭐⭐

---

### 5. 三重底/顶
- **特征**：双底/双顶的变体，三个低点/高点相近
- **含义**：比双底/双顶更扎实的反转信号
- **强度**：⭐⭐⭐⭐⭐

## 五、整理形态

### 1. 箱体整理（矩形）
```
  ───────── 上沿
  │  │ │  │
  │  │ │  │
  ───────── 下沿
```
- **特征**：价格在水平通道内波动
- **突破**：
  - 向上突破上沿 → 买点
  - 向下跌破下沿 → 卖点
- **含义**：多空平衡，等待方向选择

### 2. 上升三角形
```
     ───────── 上沿（水平）
    ／ │  │
   ／  │  │
  ／   │  │
 ───────── 下沿（上升）
```
- **特征**：上沿水平，下沿上升，波动收窄
- **含义**：买盘逐步抬高，突破概率大
- **方向**：通常向上突破
- **强度**：⭐⭐⭐⭐

### 3. 下降三角形
```
 ───────── 上沿（下降）
  ＼  │  │
   ＼ │  │
    ＼│  │
     ───────── 下沿（水平）
```
- **特征**：上沿下降，下沿水平，波动收窄
- **含义**：卖盘逐步下压，跌破概率大
- **方向**：通常向下跌破
- **强度**：⭐⭐⭐⭐

### 4. 对称三角形
```
     ───────── 上沿（下降）
    ／││
   ／ ││
  ／  ││
     ───────── 下沿（上升）
```
- **特征**：上下沿都倾斜，波动收窄
- **含义**：多空力量趋于平衡，即将选择方向
- **方向**：不确定，需等待突破确认

### 5. 旗形
```
 上升旗形：
 ─┬─
   ─┬─  ← 旗面（小幅回调）
     ─┬─
       ─┬─ ← 继续上涨

 下降旗形：
   ─┴─
 ─┴─    ← 旗面（小幅反弹）
─┴─
```
- **特征**：急涨/急跌后，出现与趋势相反的整理
- **含义**：趋势中继，整理后继续原方向
- **强度**：⭐⭐⭐⭐

## 六、缺口形态

### 缺口类型
| 类型 | 特征 | 含义 |
|-----|------|------|
| **突破缺口** | 突破重要压力/支撑，成交量放大 | 新趋势开始 |
| **中继缺口** | 趋势中途，成交量温和 | 趋势延续 |
| **衰竭缺口** | 趋势末端，成交量萎缩 | 趋势即将反转 |
| **普通缺口** | 盘整中，短期内回补 | 无特殊含义 |

### 缺口支撑/压力
- ⚠️ 缺口形成后，该价位成为重要支撑（向上缺口）或压力（向下缺口）
- 🔍 回补缺口时需关注成交量变化

## 七、实战应用要点

### 1. 形态识别优先级
```
大周期形态 > 小周期形态
多根K线形态 > 单根K线形态
颈线突破确认 > 形态初步形成
```

### 2. 成交量配合
| 形态类型 | 成交量要求 |
|---------|-----------|
| 底部反转 | 放量突破颈线 |
| 顶部反转 | 缩量跌破颈线 |
| 上升整理 | 缩量回调，放量突破 |
| 下降整理 | 缩量反弹，放量跌破 |

### 3. 止损设置
```
头肩顶/底：颈线位置 ± 3%
双底/双顶：颈线位置 ± 3%
箱体突破：箱体边界 ± 2%
单根形态：形态最低/最高价 ± 1%
```

### 4. 形态失败
- ⚠️ 约30%的形态会失败
- ✅ 必须等待突破确认后再行动
- ✅ 设置止损保护，防止形态失败造成大损失

## 八、快速识别清单

### ✅ 底部反转信号组合
- [ ] 下跌趋势末端出现
- [ ] 出现锤子线/倒锤子/看涨吞没等形态
- [ ] 成交量放大
- [ ] 次日阳线确认

### ❌ 顶部反转信号组合
- [ ] 上涨趋势末端出现
- [ ] 出现上吊线/流星线/看跌吞没等形态
- [ ] 成交量萎缩或异常放大
- [ ] 次日阴线确认

### ⏳ 趋势延续信号
- [ ] 上升旗形向上突破
- [ ] 下降旗形向下跌破
- [ ] 上升三角形向上突破
- [ ] 缺口未回补
