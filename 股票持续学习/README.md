## 三、学习计划框架

详细计划见 [LEARNING_PLAN.md](./LEARNING_PLAN.md)

### 分阶段目标

| 阶段 | 目标 | 周期 | 核心产出 |
|-----|------|------|---------|
| **L1 基础** | 掌握核心概念和方法 | Week 1-2 | 知识框架文档 |
| **L2 进阶** | 能独立分析个股 | Week 3-4 | 个股分析报告 |
| **L3 高级** | 多维度综合判断 | Week 5-6 | 综合研报 |
| **L4 实战** | 实盘验证并优化 | 持续 | 复盘记录 |

### 能力提升路径

```
T1 基本面：财报阅读 → 财务分析 → 行业对比 → 深度研报
T2 技术面：形态识别 → 均线系统 → 多周期共振 → 交易系统
T3 宏观面：指标跟踪 → 政策解读 → 周期判断 → 配置调整
T4 投资决策：估值方法 → 目标价计算 → 风险收益比 → 决策执行
T5 风险管理：规则学习 → 案例分析 → 极端预案 → 纪律执行
```

## 六、核心速查表

### T1 财报指标阈值

| 指标 | 优秀 | 良好 | 警戒 |
|-----|------|-----|------|
| ROE | >20% | 15-20% | <10% |
| 毛利率 | 行业+10% | 行业均值 | <行业-10% |
| 资产负债率 | <30% | 30-50% | >70% |
| 现金流/净利润 | >1.2 | 0.8-1.2 | <0.5 |

### T2 关键形态信号

| 形态 | 方向 | 确认条件 |
|-----|------|---------|
| 头肩顶/底 | 反转 | 突破颈线 |
| 早/黄昏之星 | 反转 | 第三根K线收盘 |
| 三白兵/乌鸦 | 反转 | 第三根K线收盘 |
| 箱体突破 | 中继 | 成交量放大 |

### T3 宏观指标信号

| 指标 | 牛市信号 | 熊市信号 |
|-----|---------|---------|
| PMI | 连续3月>50 | 连续跌破48 |
| 社融 | 增速触底回升 | 持续收缩 |
| M2 | >GDP+CPI+3% | 流动性偏紧 |
| 美联储 | 降息预期 | 激进加息 |

### T4 估值判断标准

| 方法 | 低估 | 合理 | 高估 |
|-----|------|-----|------|
| PE分位 | <30% | 30-70% | >70% |
| PB分位 | <30% | 30-70% | >70% |
| PEG | <1 | 1-1.5 | >2 |
| 股息率 | >5% | 3-5% | <1% |

### T5 风控纪律

| 触发条件 | 强制操作 |
|---------|---------|
| 单股亏损>5% | 减仓50% |
| 单股亏损>8% | 强制止损 |
| 账户亏损>10% | 仓位降至30% |
| 账户亏损>15% | 暂停新开仓 |
