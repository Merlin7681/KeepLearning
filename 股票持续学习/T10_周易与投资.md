## 一、周易三大原则与投资

### 1. 变易 — 市场永恒变化

**易理**：
> "生生之谓易" — 世界万物处于永恒变化中

**投资应用**：
| 易理 | 投资映射 | 操作启示 |
|-----|---------|---------|
| 万物皆变 | 股市永远波动 | 不预测，只应对 |
| 变中有序 | 趋势存在规律 | 顺势而为 |
| 极则必反 | 牛熊周期轮转 | 高处逃顶，低处抄底 |

**实战心法**：
```
涨久必跌，跌久必涨 — 没有只涨不跌的市场
盛极而衰，否极泰来 — 极端状态后必有反转
物极必反 — 高估值回归，低估值修复
```

---

### 2. 不易 — 投资本质不变

**易理**：
> "天行有常" — 变化中有不变的规律

**投资应用**：
| 不变规律 | 投资法则 |
|---------|---------|
| 价值规律 | 价格终将回归价值 |
| 周期规律 | 牛熊交替，周而复始 |
| 人性规律 | 贪婪与恐惧永不改变 |

**核心不变**：
```
买股票 = 买企业（格雷厄姆）
安全边际 = 永恒护城河（巴菲特）
长期持有 = 复利奇迹（芒格）
```

---

### 3. 简易 — 大道至简

**易理**：
> "易则易知，简则易从" — 简单才能执行

**投资应用**：
- 复杂策略 → 简单规则
- 多重指标 → 核心信号
- 频繁交易 → 长期持有

**简易心法**：
```
好公司 + 好价格 + 长期持有 = 财富增长
不用预测市场，只需应对市场
不用盯盘看盘，只需定期复盘
```

---

## 二、阴阳思维与市场

### 1. 阴阳对立统一

**易理**：
> "一阴一阳之谓道"

**投资映射**：
```
阳：上涨、多头、牛市、乐观、贪婪
阴：下跌、空头、熊市、悲观、恐惧

阴阳互根：没有只涨不跌，没有只跌不涨
阴阳转化：牛市转熊市，熊市转牛市
阴阳平衡：震荡市中多空博弈
```

### 2. 阳极生阴，阴极生阳

**易理**：
> "物不可以终通，故受之以否" — 泰极否来

**投资应用**：
| 阶段 | 表现 | 操作 |
|-----|------|------|
| 阳极（牛市顶峰）| 全民炒股、估值泡沫 | 减仓逃顶 |
| 阴极（熊市底部）| 哀鸿遍野、估值洼地 | 分批建仓 |
| 阴阳转换 | 趋势拐点 | 识别信号，果断行动 |

**识别信号**：
```
阳极信号：
- 成交量创历史天量
- 新开户数激增
- 垃圾股满天飞
- 全民谈股

阴极信号：
- 成交量萎缩到极点
- 老股民销户离场
- 优质股破净
- 谈股色变
```

## 四、时位观念与择时

### 1. 时 — 时机把握

**易理**：
> "时止则止，时行则行" — 顺势而为

**投资应用**：
```
牛市初期（复卦）：建仓期
牛市中期（泰卦）：持有期
牛市后期（乾卦）：减仓期
熊市初期（剥卦）：止损期
熊市中期（否卦）：空仓期
熊市后期（坤卦）：布局期
```

### 2. 位 — 位置判断

**易理**：
> "君子安其身而后动" — 位置决定策略

**投资应用**：
| 持仓位置 | 风险程度 | 操作策略 |
|---------|---------|---------|
| 底部区域 | 低风险 | 积极建仓 |
| 中位震荡 | 中风险 | 波段操作 |
| 高位区域 | 高风险 | 减仓观望 |

## 五、趋吉避凶与风险控制

### 1. 居安思危

**易理**：
> "君子终日乾乾，夕惕若厉，无咎"

**投资应用**：
- 牛市也要设止损
- 获利后分批止盈
- 永远保留现金仓位

### 2. 见微知著

**易理**：
> "履霜，坚冰至" — 见微知著

**投资应用**：
| 征兆 | 预示信号 | 应对 |
|-----|---------|------|
| 成交量萎缩 | 上攻乏力 | 准备减仓 |
| 龙头股滞涨 | 板块见顶 | 分批离场 |
| 利好不涨 | 主力出货 | 果断止损 |

### 3. 知止不殆

**易理**：
> "知止而后有定，定而后能静"

**投资应用**：
- 设定目标价，到了就卖
- 设定止损价，破了就斩
- 不贪婪，不恐惧，知止者不败
