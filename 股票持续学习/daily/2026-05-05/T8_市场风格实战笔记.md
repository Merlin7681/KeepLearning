## 六、风格类型与应对策略

### 6.1 当前风格诊断

**类型：成长牛市 / 结构分化**

特征：
- 科创半导体领涨，创业板创十年新高
- 传统板块（煤炭、银行、电力）滞涨或回调
- 估值分化严重：上证PE100%分位 vs 部分传统行业低估值

### 6.2 应对策略对照

| 风格 | 有效策略 | 无效策略 |
|------|---------|---------|
| 当前成长牛市 | 买科技/科创/成长 | 买煤炭/银行/高股息防伪 |
| 当前结构分化 | 选对板块 > 指数配置 | 均衡配置全市场 |

**仓位建议**：
- 成长仓位可达50-60%（符合当前风格）
- 煤炭/银行等传统仓位≤20%（滞涨降权）
- 保留30-40%现金（高位防调）

## 九、T8学习心得

### 实战三问（每次操作前必问）

```
□ 当前市场风格是什么？（成长/价值/防御）
□ 我的持仓与风格匹配吗？
□ 风格可能切换吗？信号是什么？
```

### 风格切换预警信号

- 🔴 从成长切价值：科技股滞涨 + 传统行业启动 + 利率上行
- 🔴 从进攻切防御：大盘高位放量滞涨 + 领涨板块补跌 + 外围动荡
- 🟢 从熊市转牛市：成交量放大 + 北向净流入 + 涨停家数>100

### 五维度速查（每日必做）

每次看盘前，10分钟内完成：
1. 看指数：上证和创业板涨跌
2. 看板块：今日涨幅前3和跌幅前3
3. 看资金：北向和两融余额变化
4. 看情绪：涨跌停家数对比
5. 定策略：调整仓位和持仓结构
