## 二、逐指标传导分析（实战核心）

### 1. GDP 到 板块配置

2026Q1 GDP同比5.0%，全年目标5%左右

传导逻辑：
- GDP 5.0% = 经济稳健 = 顺周期板块有机会
- 但增速不再加速 = 纯周期弹性有限
- 结构性机会大于总量性机会

板块传导：
- 利好 金融（银行、券商）：经济企稳=资产质量改善
- 利好 可选消费：收入预期稳定=消费信心回升
- 警惕 房地产：GDP贡献仍为负（-1%），拖累仍在
- 利好 高技术制造业：GDP贡献突出（+6.3%），新质生产力核心

实战教训：GDP是确认信号，不是领先信号。等GDP数据出来再操作已经晚了。要用PMI、社融等领先指标预判GDP方向。

---

### 2. PMI 到 趋势预判

4月PMI分化极端：高技术52.2 vs 高耗能47.9

PMI分化图：
- 高技术制造业 52.2 扩张
- 装备制造业   51.8 扩张
- 消费品行业   50.7 温和
- 综合PMI      50.1 刚站上
- --- 荣枯线 ---
- 高耗能行业   47.9 收缩

传导路径：
- 利好 高技术PMI 52.2 = 半导体、AI、新能源车景气持续
- 利好 装备制造业 51.8 = 工程机械、军工装备需求旺
- 利空 高耗能行业 47.9 = 煤炭、钢铁、水泥承压
- 中性 综合PMI 50.1 = 刚过荣枯线，经济复苏基础不牢

实战关键：PMI是先行指标，领先GDP 1-2个月。关注PMI新订单指数和出口订单。极端分化意味着：不能简单用PMI>50就看好来判断，要看结构！

---

### 3. CPI/PPI 到 通胀到货币政策的逻辑链

当前：CPI低位 + PPI刚转正 = 再通胀初期

传导链：
CPI低迷(通缩压力未消) -> 央行有宽松空间 -> 流动性充裕->利好成长股估值
PPI转正(+1.3%环比) -> 上游利润改善->资源股有机会
PPI-CPI剪刀差收窄 -> 利润从上游向中下游转移

实战传导：
- 利好 成长股：低CPI=宽松预期=DCF分母下降=估值扩张
- 利好 上游资源：PPI转正=煤炭、有色利润改善
- 利好 中游制造：PPI-CPI剪刀差收窄=成本压力减轻
- 中性 银行：低CPI=降息预期=息差承压（但经济企稳对冲）

3月PPI环比+1.3%的意义：这是PPI环比首次转正的重要信号！意味着工业品价格止跌，上游企业（如宝丰能源的煤制烯烃）利润有望改善。但要持续观察2-3个月确认趋势。

---

### 4. 社融/M2 到 流动性判断

M2增速8.5% + 社融存量456万亿

M2-GDP-CPI = 8.5% - 5.0% - ~0% = 3.5%
流动性偏宽松（大于3%阈值），资产价格上涨有基础

传导路径：
- 利好 社融放量：历史上社融大幅新增后1-3个月，A股通常见底反弹
- 利好 M2充裕：利好成长股估值扩张
- 关注 居民贷款：如果居民不借钱，社融放量效果打折

实战关键：社融是市场底的领先指标。历史规律：社融触底=1-3个月后A股见底。但要区分谁在借钱：企业中长期贷款(投资) > 居民贷款(消费/购房) > 短期贷款(套利)

---

### 5. 美联储/美国数据 到 外部约束

美国CPI 3.3% + 核心CPI 2.6% + GDP 2.7% + 失业率4.3%

传导链：
美国经济软着陆但通胀粘性 -> 美联储降息节奏延后 -> 美元维持强势 -> 人民币贬值压力 -> 外资流出压力 -> A股核心资产承压

传导路径：
- 利空 美联储不降息：美元强势=人民币承压=外资流出
- 利空 中美利差倒挂：国内货币政策空间受限
- 利好 美国GDP 2.7%韧性：中国出口不差=外需有支撑
- 中性 关税风险：出口结构分化，对新兴市场出口强，对美弱

实战教训：美联储是全球资产定价锚。A股不能只看国内，必须盯美联储！美联储降息=A股流动性改善；不降息=成长股估值天花板受限。

## 四、宏观指标实战心法

### 心法1：领先 vs 同步 vs 滞后

领先指标（预判未来1-3个月）：PMI -> 社融 -> M2 -> 出口订单
同步指标（确认当前状态）：GDP -> 工业增加值 -> 社零
滞后指标（确认已发生趋势）：CPI -> PPI -> 失业率

实战：用领先指标做决策，用同步指标确认，用滞后指标纠错

### 心法2：极端值比趋势值更重要

PMI 50.1(温和) -> 不操作，观察
PMI 47.9(高耗能极端收缩) -> 回避相关板块
PMI 52.2(高技术极端扩张) -> 加仓相关板块

教训：平均数会掩盖结构性分化！

### 心法3：数据与市场的关系

数据好于预期 -> 利好兑现 -> 可能高开低走
数据差于预期 -> 政策刺激预期 -> 可能低开高走
数据符合预期 -> 无特殊影响

实战：不是数据好就买数据差就卖，而是数据差->政策预期->买点（逆向思维）

### 心法4：中国宏观的特殊性

中国宏观政策是逆周期调节：
经济差 -> 政策加码 -> 市场反而可能涨
经济好 -> 政策退出 -> 市场反而可能跌

关键跟踪：
政治局会议（季度）-> 定调
央行货币政策执行报告 -> 执行
LPR每月20日 -> 信号
