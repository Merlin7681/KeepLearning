## 一、兴业银行：PB估值法实战

### 1. 基础数据

| 指标 | 数值 | 来源 |
|------|------|------|
| 股价 | 17.93元 | 4/30收盘 |
| PE(TTM) | 4.90倍 | 实时行情 |
| PB | 0.46倍 | 实时行情 |
| 股息率 | 9.06% | 实时行情 |
| 每股净资产 | 39.34元 | 2026Q1财报 |
| ROE(TTM) | 8.49% | 财务数据 |
| 年初至今 | -12.52% | 行情数据 |

### 2. PB估值法核心逻辑

**银行股为什么用PB？**

- 银行是重资产行业，净资产是核心
- PE受周期波动影响大（盈利高点PE低，盈利低点PE高）
- PB更稳定，反映净资产溢价

**PB估值公式**：
```
目标价 = 每股净资产 × 合理PB
```

### 3. 合理PB的确定（三方法）

#### 方法一：历史分位法

兴业银行PB历史数据：
- 近5年PB区间：0.4-1.2倍
- 当前PB 0.46倍，处于历史5.13%分位（极度低估）
- 历史中位数：约0.7倍

**结论**：按历史中位数0.7倍算，目标价 = 39.34 × 0.7 = 27.54元

#### 方法二：ROE联动法

**PB与ROE的关系**：
| ROE水平 | 合理PB区间 |
|--------|-----------|
| 5% | 0.5-1倍 |
| 10% | 1-1.5倍 |
| 15% | 1.5-2.5倍 |

兴业银行ROE 8.49%，介于5%-10%之间
按线性插值：合理PB ≈ 0.8-1.2倍

**但考虑因素**：
- 银行股整体估值下移（利率下行、息差收窄）
- 股份制银行合理PB：0.6-1.0倍
- 兴业银行作为绿色金融龙头，可给上限：0.8倍

**结论**：按ROE联动法，目标价 = 39.34 × 0.8 = 31.47元

#### 方法三：行业对比法

股份制银行PB对比（2026年4月）：
| 银行 | PB | ROE | 股息率 |
|------|----|----|--------|
| 招商银行 | 0.91 | 高 | 7.65% |
| 兴业银行 | 0.46 | 中 | 9.06% |
| 光大银行 | 0.38 | 中 | 6.01% |
| 民生银行 | 0.29 | 低 | 5.29% |

兴业银行PB 0.46，低于招行0.91，但股息率更高（9.06% vs 7.65%）
合理PB应介于招行和光大之间：0.6-0.8倍

**结论**：按行业对比法，目标价 = 39.34 × 0.7 = 27.54元

### 4. 综合目标价

| 方法 | 合理PB | 目标价 | 权重 |
|------|--------|--------|------|
| 历史分位法 | 0.7倍 | 27.54元 | 30% |
| ROE联动法 | 0.8倍 | 31.47元 | 40% |
| 行业对比法 | 0.7倍 | 27.54元 | 30% |
| **综合** | **0.73倍** | **28.72元** | — |

### 5. 风险收益比计算

```
当前价：17.93元
目标价：28.72元（+60.2%）
止损价：16.00元（PB 0.4倍，历史极端低位）

风险收益比 = (28.72 - 17.93) / (17.93 - 16.00)
           = 10.79 / 1.93
           = 5.6:1
```

**结论**：风险收益比5.6:1，优秀！但需注意：
- 银行股修复需要时间（以年为单位）
- 适合长期持有吃股息（9.06%），等待估值修复

### 6. 兴业银行估值总结

| 维度 | 结论 |
|------|------|
| **PB估值** | 0.46倍处于历史5%分位，极度低估 |
| **目标价** | 28.72元（+60.2%空间） |
| **股息率** | 9.06%，已具备持有价值 |
| **风险收益比** | 5.6:1，优秀 |
| **操作建议** | **买入**，长期持有，股息+估值修复双击 |

## 二、宝丰能源：PE/PEG估值法实战

### 1. 基础数据

| 指标 | 数值 | 来源 |
|------|------|------|
| 股价 | 30.70元 | 4/30收盘 |
| PE(TTM) | 17.90倍 | 实时行情 |
| PB | 4.59倍 | 实时行情 |
| 股息率 | 2.26% | 实时行情 |
| 每股收益(2025) | 1.56元 | 2025年报 |
| ROE(2025) | 24.84% | 2025年报 |
| 净利润增速(2025) | 79.09% | 同比 |
| 3年复合增速 | 21.66% | CAGR |
| 年初至今 | +59.81% | 行情数据 |

### 2. PE估值法

**PE估值公式**：
```
目标价 = EPS × 合理PE
```

#### 合理PE的确定

**方法一：历史分位法**
- 当前PE 17.90倍，处于历史56%分位
- 历史中位数约16倍
- 但公司已从成长期进入成熟期，PE中枢下移

**方法二：增速匹配法（PEG）**

宝丰能源增速分析：
- 2025年增速79.09%（高增长，但不可持续）
- 3年复合增速21.66%
- 2026年预期增速：机构预测约15-20%（产能释放）

**PEG计算**：
```
当前PE = 17.90倍
预期增速 = 18%（取中值）
PEG = 17.90 / 18 = 0.99
```

**PEG判断**：

- PEG < 1：低估
- PEG = 1：合理
- PEG > 1：高估

当前PEG 0.99，处于合理偏低区间！

**方法三：行业对比法**

煤化工行业PE对比：
| 公司 | PE | ROE | 增速 |
|------|----|----|------|
| 宝丰能源 | 17.9 | 24.84% | 高 |
| 中国神华 | 12.6 | 中 | 低 |
| 陕西煤业 | 10.2 | 中 | 低 |

宝丰能源ROE远高于行业，增速也更高，PE溢价合理。

### 3. 综合目标价（PE法）

| 方法 | 合理PE | 目标价 | 权重 |
|------|--------|--------|------|
| 历史分位法 | 16倍 | 24.96元 | 20% |
| PEG法 | 18倍 | 28.08元 | 50% |
| 行业对比法 | 20倍 | 31.20元 | 30% |
| **综合** | **18.2倍** | **28.39元** | — |

### 4. 风险收益比计算

```
当前价：30.70元
目标价：28.39元（-7.5%）
止损价：27.00元（PE 15倍，历史低位）

风险收益比 = (28.39 - 30.70) / (30.70 - 27.00)
           = -2.31 / 3.70
           = -0.6:1（负值！）
```

**结论**：当前价30.70元已高于目标价28.39元，估值合理偏高！

### 5. 宝丰能源估值陷阱警示

#### 陷阱1：增速不可持续

2025年增速79.09%，但：
- 内蒙古项目达产，高增速难再现
- 2026年预期增速降至15-20%
- PEG计算需用未来增速，不能用历史增速

**修正**：用2026年预期增速18%算PEG = 0.99，合理

#### 陷阱2：周期股估值反直觉

煤化工是周期行业：
- 盈利高点 → PE看似低 → 实际高估
- 盈利低点 → PE看似高 → 实际低估

当前处于盈利高点，PE 17.9看似不高，但需警惕周期回落。

#### 陷阱3：PB过高

PB 4.59倍，远高于行业（煤炭股PB多在1-2倍）
- ROE 24.84%支撑高PB
- 但ROE回落风险需警惕

### 6. 宝丰能源估值总结

| 维度 | 结论 |
|------|------|
| **PE估值** | 17.9倍，PEG 0.99，合理偏低 |
| **目标价** | 28.39元（-7.5%空间） |
| **风险点** | 增速回落、周期高点、PB偏高 |
| **风险收益比** | -0.6:1，不建议追高 |
| **操作建议** | **观望**，等回调至28元以下再考虑 |

## 三、两股对比：估值方法论总结

### 1. 银行股 vs 成长股：估值方法选择

| 类型 | 适用方法 | 原因 |
|------|---------|------|
| **银行股** | PB法 | 重资产、盈利周期波动、净资产稳定 |
| **成长股** | PEG法 | 增速是核心，PE需与增速匹配 |
| **周期股** | PB法 | 盈利高点PE低、低点PE高，PB更可靠 |
| **高股息股** | 股息率法 | 分红收益率是核心 |

### 2. 兴业银行 vs 宝丰能源：估值结果对比

| 维度 | 兴业银行 | 宝丰能源 |
|------|---------|---------|
| **估值方法** | PB法 | PE/PEG法 |
| **当前估值** | PB 0.46（极度低估） | PE 17.9（合理） |
| **目标价** | 28.72元 | 28.39元 |
| **上涨空间** | +60.2% | -7.5% |
| **风险收益比** | 5.6:1（优秀） | -0.6:1（差） |
| **股息率** | 9.06% | 2.26% |
| **操作建议** | **买入** | **观望** |

### 3. 估值实战心法

#### 心法1：方法匹配行业

```
银行/保险/券商 → PB法
成长股 → PEG法
周期股 → PB法（或反向用PE）
高股息股 → 股息率法
现金流稳定股 → DCF法
```

#### 心法2：多方法交叉验证

单一方法易出错，需多方法验证：
- 兴业银行：历史分位+ROE联动+行业对比，三方法一致指向低估
- 宝丰能源：PEG看似合理，但风险收益比为负，提示高估

#### 心法3：警惕估值陷阱

| 陷阱 | 表象 | 真相 | 案例 |
|-----|------|------|------|
| **周期陷阱** | PE很低 | 盈利高点，实际高估 | 周期股顶点 |
| **增速陷阱** | PEG合理 | 增速不可持续 | 宝丰能源 |
| **破净陷阱** | PB<1 | 资产质量恶化 | 需看ROE |
| **高股息陷阱** | 股息率很高 | 分红不可持续 | 地产股暴雷前 |

#### 心法4：估值是区间，不是精确值

```
估值是艺术，不是科学
目标价是区间：28.72元 ± 10%
不要追求精确，要追求方向正确
```
