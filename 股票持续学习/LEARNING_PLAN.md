## 二、执行机制

### 每日任务（盘后）

**⚠️ 重要机制：每次学习前，先分析市场风格！**

```
【市场风格速查】（每日必做）
□ 大势：牛/熊/震荡？
□ 领涨板块：_____
□ 领跌板块：_____
□ 北向资金：净流入/流出 ____亿
□ 情绪：贪婪/中性/恐慌
□ 风格判断：成长/价值/防御
□ 今日策略：_____
```

完成风格分析后，再进行以下学习：

- [ ] 阅读当日宏观要闻（10分钟）
- [ ] 观察池标的盘后复盘（重点关注异动）
- [ ] 记录学习笔记到 memory/YYYY-MM-DD.md
- [ ] 更新市场风格跟踪表（周度）

### 每周任务（周末）
- [ ] 深度分析1只股票（输出完整研报）
- [ ] 复盘本周决策，更新经验到 MEMORY.md
- [ ] 阅读指定学习资源1章节

### 每月任务
- [ ] 更新行业估值中枢数据
- [ ] 复盘学习计划执行情况
- [ ] 调整下月学习重点
