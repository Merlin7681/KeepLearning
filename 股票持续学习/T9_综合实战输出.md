## 一、综合实战输出框架

### 1.1 标准化研报模板

一份完整的个股深度研报应该包含以下核心模块：

```
【标题】
  公司名称+代码+评级（如：招商银行(600036)深度报告-强烈推荐）

【第一部分：投资要点】（300字内）
  1. 核心投资逻辑（一句话概括）
  2. 关键数据亮点
  3. 目标价与上涨空间
  4. 主要风险提示

【第二部分：公司概况】（500字）
  - 公司主营业务
  - 行业地位
  - 核心竞争力（护城河）

【第三部分：财务分析】（800字）
  - 盈利能力（ROE/毛利率/净利率）
  - 成长性（营收/净利增速）
  - 财务安全（负债率/现金流/商誉）
  - 估值（PE/PB/股息率）

【第四部分：行业分析】（600字）
  - 行业空间与增速
  - 竞争格局
  - 政策环境
  - 周期位置

【第五部分：技术分析】（400字）
  - 趋势判断（均线/MACD）
  - 关键价位（支撑/压力）
  - 成交量配合

【第六部分：资金面】（300字）
  - 主力资金流向
  - 股东结构
  - 机构持仓

【第七部分：风险提示】（300字）
  - 行业风险
  - 公司风险
  - 政策风险
  - 技术风险

【第八部分：投资建议】（200字）
  - 评级
  - 目标价
  - 止损位
  - 仓位建议
  - 操作节奏
```

【关键数据】

## 二、分析框架检查清单

### 2.1 T1-T5 必备检查项

| 维度 | 必须分析项 | 优先级 |
|------|-----------|---------|
| **T1基本面** | ROE、毛利率、净利率、营收增速、净利润增速、现金流/净利、资产负债率 | 必须 |
| **T2技术面** | 均线趋势、MACD信号、KDJ位置、布林带、支撑压力位 | 必须 |
| **T3宏观面** | 行业周期、宏观经济、政策影响 | 必须 |
| **T4估值** | PE/PB历史分位、机构目标价、PEG、股息率 | 必须 |
| **T5风控** | 止损位、仓位建议、风险清单 | 必须 |

### 2.2 快速检查项（10分钟）

如果时间紧迫，至少确认以下5项：

```
□ 当前股价 vs 机构目标价（有上涨空间？）
□ 当前PE历史分位（高估/低估？）
□ 均线趋势（多头/空头？）
□ 支撑/止损位（风险可控？）
□ 行业逻辑（逻辑通顺？）
```

## 三、输出格式规范

### 3.1 数字格式

| 项目 | 格式 | 示例 |
|------|------|------|
| 股价 | XX.XX元 | 17.93元 |
| 市值 | XX.XX亿 | 3,794.00亿 |
| PE/PB | XX.XX倍 | 4.90倍 |
| 增速 | +XX.X% | +45.64% |
| ROE | XX.XX% | 24.84% |
| 毛利率 | XX.XX% | 35.92% |
| 股息率 | X.XX% | 9.06% |

### 3.2 评级规范

| 评级 | 定义 | 预期涨幅 |
|------|------|---------|
| **强烈推荐** | 确定性高，向上空间 > 50% | >50% |
| **推荐/买入** | 胜率较高，空间 20-50% | 20-50% |
| **增持** | 胜率中等，空间 10-20% | 10-20% |
| **中性/持有** | 空间有限，持有观察 | <10% |
| **减持/回避** | 风险大于机会 | 负收益 |

### 3.3 目标价设置规范

```
目标价 = 合理估值 × 调整系数

合理估值计算方法：
  1. 历史PE分位法（PE中位数 × 合理分位）
  2. 机构一致性预期（目标价均值）
  3. 相对行业法（行业平均PE × 公司增速溢价）

调整系数：
  - 确定性高：1.0-1.1
  - 不确定性：0.8-0.9
  - 风险较大：0.7以下
```

## 四、研报质量自检表

### 4.1 基础自检（完成度）

```
□ 有标题和评级
□ 有核心观点（3句话内概括）
□ 有关键财务数据（营收/净利/ROE/PE）
□ 有行业分析
□ 有技术分析
□ 有风险提示
□ 有操作建议（目标价/止损位/仓位）
```

### 4.2 进阶自检（专业度）

```
□ 财务数据来源标注（年报/季报/公告）
□ 估值方法明确（PE/PB/DCF/股息率）
□ 行业对比数据（公司在行业中排名）
□ 横向与纵向对比（vs行业平均、vs历史）
□ 风险量化（下跌空间 vs 上涨空间）
□ 操作有具体价位（不只是"持有"）
```

### 4.3 决策自检（实用性）

```
□ 回答了"买不买"这个问题
□ 给出了具体买入价位
□ 给出了止损价位
□ 仓位建议清晰
□ 逻辑链条完整（数据→分析→结论）
```

## 五、案例：标准化研报输出示范

### 兴业银行深度研报模板

```
【标题】
  兴业银行(601166)深度报告-推荐
  评级：推荐 | 目标价：24元 | 当前价：17.93元 | 上涨空间：33.8%

【核心观点】（3句话）
  1. 银行板块被极端低估，PE 4.9倍处于历史5%分位，PB 0.46倍破净56%
  2. 资产质量稳健，不良率1.08%，拨备覆盖率224%，安全垫充足
  3. 股息率9.06%创历史新高，高股息策略优选

【关键数据】
  ┌─────────────┬─────────┬─────────┐
  │ 指标        │ 2025年  │ 同比    │
  ├─────────────┼─────────┼─────────┤
  │ 营收        │ 2127亿   │ +0.24% │
  │ 净利        │ 775亿    │ +0.34% │
  │ ROE         │ 9.15%   │ -0.3pct │
  │ 不良率      │ 1.08%   │ 持平    │
  │ 拨备覆盖率  │ 224%     │ -5pct   │
  │ PE(TTM)    │ 4.9倍   │ 5%分位  │
  │ PB          │ 0.46倍  │ 极端低   │
  │ 股息率      │ 9.06%   │ 历史新高 │
  └─────────────┴─────────┴─────────┘

【行业分析】
  - 银行板块PB 0.46倍，极端破净
  - 净息差1.46%处历史低位，继续下行空间有限
  - 政策持续宽松托底经济，利好银行资产质量

【技术分析】
  - 均线：5日>10日>20日，空头排列但接近底部
  - MACD：零轴下方运行，等待金叉信号
  - KDJ：K=13.5，D=13.6，超卖区
  - 支撑位：17元（前期低点）
  - 压力位：18.5元（20日均线）

【资金面】
  - 主力资金：近5日净流出收窄
  - 股东：国家队+险资主导，筹码稳定

【风险提示】
  ⚠️ 净息差继续收窄风险
  ⚠️ 经济复苏不及预期
  ⚠️ 房地产敞口风险

【操作建议】
  评级：推荐
  目标价：24元（上涨空间33.8%）
  止损位：16元（止损幅度-10.8%）
  建议仓位：10-15%
  建仓节奏：
    - 首仓：17元以下建仓30%
    - 加仓：16元以下加仓40%
    - 满仓：15元以下打满30%
```

## 六、快速输出检查清单

### 输出前自检（1分钟）

```
□ 核心观点用3句话说清楚了吗？
□ 关键数据（营收/净利/ROE/PE）列出来了吗？
□ 目标价和止损价给出来了吗？
□ 仓位建议有吗？
□ 风险说清楚了吗？
```

### 输出后复核（1分钟）

```
□ 逻辑链条通顺吗？（数据→分析→结论）
□ 数字单位统一了吗？（亿元/万元/元）
□ 评级和目标价对应吗？（推荐→有上涨空间）
□ 止损位设了吗？（风控是底线）
```
