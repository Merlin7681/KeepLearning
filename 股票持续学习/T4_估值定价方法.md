## 一、估值方法总览

| 方法 | 适用场景 | 核心逻辑 | 难度 |
|-----|---------|---------|------|
| **PE估值** | 盈利稳定的公司 | 多少年回本 | ⭐ |
| **PB估值** | 重资产、周期股 | 净资产溢价 | ⭐ |
| **PEG估值** | 成长股 | 估值与增速匹配 | ⭐⭐ |
| **DCF估值** | 现金流可预测的公司 | 未来现金流折现 | ⭐⭐⭐⭐ |
| **股息率估值** | 高分红公司 | 分红收益率 | ⭐ |

## 二、PE估值法（市盈率）

### 1. 基本公式

```
目标价 = EPS × 合理PE
```

- **EPS**：每股收益（用最近12个月净利润÷总股本）
- **合理PE**：根据历史分位、行业均值、增速综合判断

### 2. 合理PE的确定

#### 方法一：历史分位法

```
合理PE = PE历史均值（或中位数）
```

**判断标准**：
| 当前PE分位 | 估值判断 | 操作建议 |
|-----------|---------|---------|
| <20% | 低估 | 买入区间 |
| 20-50% | 合理偏低 | 可建仓 |
| 50-80% | 合理偏高 | 减持 |
| >80% | 高估 | 卖出 |

**示例**：
```
茅台近5年PE区间：25-50倍
当前PE：32倍
分位位置：40%左右
结论：合理偏低，可持有
```

#### 方法二：行业对比法

```
合理PE = 行业均值PE × 个股质量系数
```

**质量系数调整**：
| 因素 | 调整方向 |
|-----|---------|
| ROE高于行业+5% | PE+2倍 |
| 市占率行业第一 | PE+3倍 |
| 品牌溢价强 | PE+2倍 |
| 负债率高于行业10% | PE-3倍 |
| 增速低于行业 | PE-2倍 |

#### 方法三：增速匹配法（PEG）

```
合理PE = 盈利增速 × PEG系数
```

- PEG < 1：低估
- PEG = 1：合理
- PEG > 1：高估

**示例**：
```
某公司预期增速20%
PEG取1（合理估值）
合理PE = 20倍
当前PE = 25倍
结论：高估25%，不建议买入
```

### 3. PE估值的陷阱

| 情况 | PE表现 | 实际含义 |
|-----|-------|---------|
| **周期股底部** | PE看似很高 | 实际低估（盈利低点） |
| **周期股顶部** | PE看似很低 | 实际高估（盈利高点） |
| **亏损公司** | PE为负 | 不适用PE估值 |
| **一次性收益** | PE虚低 | 需用扣非PE |

**周期股用PB更准确！**

## 三、PB估值法（市净率）

### 1. 基本公式

```
目标价 = 每股净资产 × 合理PB
```

### 2. 适用场景

- ✅ 银行、保险、券商（金融业）
- ✅ 钢铁、煤炭、有色（周期股）
- ✅ 地产、基建（重资产行业）
- ⚠️ 科技、消费（轻资产，PB参考意义有限）

### 3. 合理PB的确定

| PB水平 | 含义 | 判断 |
|-------|------|------|
| <1 | 破净 | 可能被低估或资产质量差 |
| 1-2 | 低估值 | 安全边际高 |
| 2-4 | 合理区间 | 正常水平 |
| 4-6 | 偏高 | 需要验证成长性 |
| >6 | 高估 | 需要极高ROE支撑 |

### 4. PB与ROE联动

```
合理PB = f(ROE)
```

| ROE水平 | 合理PB区间 |
|--------|-----------|
| 5% | 0.5-1倍 |
| 10% | 1-1.5倍 |
| 15% | 1.5-2.5倍 |
| 20% | 2.5-4倍 |
| 25%+ | 4倍+ |

**核心逻辑**：ROE越高，净资产创造价值能力越强，PB溢价越高。

### 5. 银行股PB估值模板

```
国有大行合理PB：0.5-0.8倍
股份制银行合理PB：0.6-1.0倍
城商行合理PB：0.8-1.2倍
```

**示例**：
```
招商银行每股净资产：36元
合理PB：0.8倍
目标价 = 36 × 0.8 = 28.8元
当前价：34元
结论：估值合理偏高，等待回调
```

## 四、PEG估值法

### 1. 基本公式

```
PEG = PE / 盈利增速
```

### 2. 判断标准

| PEG值 | 估值判断 |
|-------|---------|
| <0.5 | 严重低估 |
| 0.5-0.8 | 低估 |
| 0.8-1.2 | 合理 |
| 1.2-2 | 偏高 |
| >2 | 高估 |

### 3. 适用场景

- ✅ 成长股（增速15-40%）
- ⚠️ 低增速股（PEG失效，用PE）
- ❌ 亏损股（PEG为负，不适用）

### 4. 实战应用

**示例**：
```
宁德时代当前PE：30倍
预期3年复合增速：25%
PEG = 30 / 25 = 1.2
结论：合理偏高，成长性匹配

比亚迪当前PE：45倍
预期3年复合增速：35%
PEG = 45 / 35 = 1.28
结论：合理偏高，成长性支撑估值
```

## 五、DCF估值法（现金流折现）

### 1. 核心逻辑

```
企业价值 = Σ 未来现金流 / (1+折现率)^n
```

### 2. 简化版DCF公式

```
目标价 = (FCF × (1+g)) / (r - g) ÷ 总股本
```

- **FCF**：自由现金流（经营现金流 - 资本支出）
- **g**：永续增长率（通常取2-4%）
- **r**：折现率（通常取8-12%）

### 3. 参数确定

| 参数 | 取值建议 | 依据 |
|-----|---------|------|
| **折现率r** | 8-12% | 无风险利率+风险溢价 |
| **永续增长g** | 2-4% | 接近GDP增速 |
| **预测期** | 5-10年 | 行业可预测性 |

### 4. 敏感性分析

**示例**：某公司FCF=10亿，股本10亿股

| 折现率r | 永续增长g | 每股价值 |
|--------|----------|---------|
| 10% | 3% | 14.3元 |
| 10% | 4% | 16.7元 |
| 8% | 3% | 20.4元 |
| 12% | 3% | 11.1元 |

**结论**：折现率每变动1%，估值变动10-20%，非常敏感！

### 5. DCF适用场景

- ✅ 现金流稳定的公司（公用事业、消费龙头）
- ⚠️ 周期股（现金流波动大，预测困难）
- ❌ 初创公司（无稳定现金流）

## 六、股息率估值法

### 1. 基本公式

```
合理股价 = 年度分红 / 目标股息率
```

### 2. 目标股息率确定

| 投资类型 | 目标股息率 |
|---------|-----------|
| **高股息策略** | ≥5% |
| **稳健配置** | 3-5% |
| **成长配置** | <3%（分红少） |

### 3. 实战应用

**示例**：
```
中国神华年度分红：2.5元/股
目标股息率：5%
合理股价 = 2.5 / 0.05 = 50元
当前价：45元
股息率 = 2.5 / 45 = 5.56%
结论：股息率达标，可配置
```

## 七、目标价计算模板

### 单一方法计算

```
【PE法】
当前EPS：3.5元
合理PE：15倍（历史30%分位）
目标价 = 3.5 × 15 = 52.5元

【PB法】
每股净资产：20元
合理PB：1.5倍
目标价 = 20 × 1.5 = 30元

【PEG法】
预期增速：20%
PEG取1
合理PE = 20倍
目标价 = EPS × 20
```

### 多方法综合

| 方法 | 目标价 | 权重 |
|-----|-------|------|
| PE法 | 52.5元 | 40% |
| PB法 | 30元 | 30% |
| PEG法 | 45元 | 30% |
| **综合目标价** | **44.1元** | — |

## 八、止损价计算方法

### 1. 技术位止损

| 方法 | 计算方式 | 适用场景 |
|-----|---------|---------|
| **固定比例止损** | 买入价 × (1-8%) | 短线交易 |
| **支撑位止损** | 近期低点下方3% | 技术派 |
| **均线止损** | 跌破20日线止损 | 趋势交易 |
| **颈线止损** | 形态颈线下方3% | 形态交易 |

### 2. 估值止损

```
止损价 = 估值下限
```

**示例**：
```
PE历史底部：12倍
EPS：3元
估值止损价 = 3 × 12 = 36元
当前价：40元
止损空间：-10%
```

### 3. 止损执行纪律

| 亏损幅度 | 强制操作 |
|---------|---------|
| >5% | 减仓50% |
| >8% | 强制止损 |
| 触及止损价 | 无条件执行，不幻想反弹 |

## 九、风险收益比评估

### 1. 基本公式

```
风险收益比 = (目标价 - 当前价) / (当前价 - 止损价)
```

### 2. 判断标准

| 风险收益比 | 评价 | 操作建议 |
|-----------|------|---------|
| >3:1 | 优秀 | 重仓 |
| 2-3:1 | 良好 | 标配 |
| 1-2:1 | 一般 | 轻仓 |
| <1:1 | 较差 | 不操作 |

### 3. 实战应用

**示例**：
```
当前价：40元
目标价：52元（+30%）
止损价：37元（-7.5%）
风险收益比 = (52-40) / (40-37) = 4:1
结论：风险收益比优秀，可建仓
```

## 十一、估值陷阱警示

### 1. 价值陷阱

| 表象 | 真相 | 案例 |
|-----|------|------|
| PE很低 | 盈利不可持续 | 周期股顶点 |
| PB很低 | 资产质量恶化 | 银行不良率高 |
| 股息率很高 | 分红不可持续 | 地产股暴雷前 |

### 2. 成长陷阱

| 表象 | 真相 | 案例 |
|-----|------|------|
| 增速很高 | 增速即将回落 | 概念股炒作 |
| PEG合理 | 增速预测过于乐观 | 研究员一致预期 |
| 故事很好 | 无法兑现 | 乐视网 |

### 3. 周期陷阱

```
周期股估值铁律：
- PE高点买入（盈利低点）
- PE低点卖出（盈利高点）
- PB更可靠：周期底部PB接近1倍
```
