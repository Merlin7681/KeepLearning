## 一、核心理念：治未病

### 1. 上医治未病

**内经原文**：
> "上医治未病，中医治欲病，下医治已病"

**投资映射**：
| 医者 | 投资对应 | 核心能力 |
|-----|---------|---------|
| 上医 | 顶级投资者 | 预防风险，提前布局 |
| 中医 | 成熟投资者 | 发现问题，及时调整 |
| 下医 | 普通投资者 | 问题爆发，被动应对 |

**应用心法**：
```
最好的风控 = 不让风险发生
最好的抄底 = 在恐慌前布局
最好的止损 = 买入前就想好退路
```

---

### 2. 治未病三层次

| 层次 | 内经理念 | 投资应用 |
|-----|---------|---------|
| **未病先防** | 养生保健 | 选股前严格筛选，排除问题股 |
| **欲病早治** | 早期干预 | 发现苗头及时止损，不等恶化 |
| **已病防变** | 防止传变 | 暴雷后快速反应，防止损失扩大 |

**投资体检表**：
```
买入前（未病先防）：
□ 财务健康度评分 > 70
□ 大股东质押 < 30%
□ 商誉占比 < 20%
□ 现金流/净利润 > 0.8

持有中（欲病早治）：
□ 季报指标有无恶化
□ 行业政策有无利空
□ 技术面是否破位

突发风险（已病防变）：
□ 立即评估影响程度
□ 制定应对方案
□ 果断执行，不加仓摊成本
```

## 二、阴阳理论与资产配置

### 1. 阴阳平衡

**内经原文**：
> "阴平阳秘，精神乃治"

**投资映射**：
```
阳：高风险资产（成长股、科技股）
阴：低风险资产（债券、高股息股）

阴阳平衡 → 资产配置均衡
阳盛阴虚 → 激进配置（牛市后期危险）
阴盛阳虚 → 保守配置（熊市后期机会）
```

### 2. 动态平衡配置

| 市场环境 | 阴阳状态 | 配置建议 |
|---------|---------|---------|
| 牛市初期 | 阴极生阳 | 30%防御 + 70%进攻 |
| 牛市中期 | 阳气上升 | 40%防御 + 60%进攻 |
| 牛市后期 | 阳极生阴 | 60%防御 + 40%进攻 |
| 熊市中期 | 阴气弥漫 | 70%防御 + 30%进攻 |
| 熊市后期 | 阴极生阳 | 增加进攻仓位 |

**动态调整原则**：
```
根据市场估值分位调整：
- 估值 < 30%分位 → 加大进攻仓位
- 估值 30-70%分位 → 维持平衡
- 估值 > 70%分位 → 加大防御仓位
```

## 三、四时养生与投资周期

### 1. 四时养生

**内经原文**：
> "春生、夏长、秋收、冬藏，是谓四时之序"

**投资映射**：
| 季节 | 养生 | 投资策略 | 市场阶段 |
|-----|------|---------|---------|
| **春** | 生发 | 播种布局 | 熊市见底，开始建仓 |
| **夏** | 繁茂 | 持有增长 | 牛市上涨，坚定持有 |
| **秋** | 收敛 | 收获止盈 | 牛市见顶，分批减仓 |
| **冬** | 闭藏 | 休养生息 | 熊市下跌，现金为王 |

### 2. 顺势而为

**内经原文**：
> "春三月，此谓发陈...养生之道也"

**投资应用**：
```
春（复苏期）：
- 选优质种子股
- 分批建仓
- 耐心等待发芽

夏（繁荣期）：
- 持股为主
- 借势上涨
- 不轻易下车

秋（调整期）：
- 分批收获
- 落袋为安
- 控制仓位

冬（萧条期）：
- 现金为王
- 观察研究
- 准备春季播种
```

## 四、正气存内与投资心态

### 1. 正气存内

**内经原文**：
> "正气存内，邪不可干"

**投资映射**：
```
正气 = 投资原则、交易系统、心理定力
邪气 = 市场噪音、情绪波动、贪婪恐惧

正气存内 → 坚持原则，不受干扰
邪气干之 → 追涨杀跌，情绪化交易
```

### 2. 养正气之法

| 养生 | 投资修炼 |
|-----|---------|
| 调息养神 | 保持平和心态 |
| 饮食有节 | 信息过滤，不看垃圾消息 |
| 起居有常 | 交易纪律，严格执行计划 |
| 不妄作劳 | 不过度交易，控制频率 |
| 精神内守 | 专注自己策略，不攀比 |

### 3. 七情调和

**内经原文**：
> "怒伤肝、喜伤心、思伤脾、悲伤肺、恐伤肾"

**投资情绪管理**：
| 情绪 | 投资表现 | 对策 |
|-----|---------|------|
| **怒** | 亏损后愤怒加仓 | 冷静24小时再决策 |
| **喜** | 获利后过度乐观 | 设定止盈，不追高 |
| **思** | 过度分析犹豫不决 | 简化策略，执行规则 |
| **悲** | 亏损后绝望离场 | 相信周期，坚持计划 |
| **恐** | 恐慌性抛售 | 坚持止损纪律，不加杠杆 |

## 五、整体观与宏观分析

### 1. 天人合一

**内经原文**：
> "人与天地相参，与日月相应"

**投资映射**：
```
天 = 宏观经济、货币政策
人 = 企业经营、行业周期

投资要"天人合一"：
- 宏观环境与微观企业共振
- 政策周期与产业周期同步
- 经济周期与企业盈利匹配
```

### 2. 五行生克与行业轮动

| 五行 | 对应行业 | 生克关系 |
|-----|---------|---------|
| **木** | 科技、医药 | 木生火 → 科技带火消费 |
| **火** | 消费、教育 | 火生土 → 消费拉动地产 |
| **土** | 地产、基建 | 土生金 → 地产带动金融 |
| **金** | 金融、机械 | 金生水 → 金融支持制造 |
| **水** | 公用事业、物流 | 水生木 → 基础设施支撑创新 |

**行业轮动逻辑**：
```
经济复苏 → 金融先行（金）
经济繁荣 → 消费医药（火）
经济过热 → 资源周期（土）
经济衰退 → 公用事业（水）
政策刺激 → 科技创新（木）
```

## 六、治病求本与投资本质

### 1. 治病求本

**内经原文**：
> "治病必求于本"

**投资映射**：
```
本 = 企业内在价值
标 = 股价短期波动

投资要"治病求本"：
- 看透企业本质，不被表面现象迷惑
- 关注长期价值，忽略短期噪音
- 理解商业模式，不追逐概念炒作
```

### 2. 辨证论治

| 病症 | 投资问题 | 辨证要点 | 治疗方案 |
|-----|---------|---------|---------|
| 虚证 | 业绩下滑 | 查看现金流是否虚 | 剔除伪成长股 |
| 实证 | 估值过高 | 判断泡沫程度 | 高估减持 |
| 寒证 | 行业衰退 | 识别周期位置 | 等待周期反转 |
| 热证 | 概念炒作 | 分清真假热度 | 避开纯炒作 |

### 2. 风控清单

```
事前风控：
□ 排除财务造假嫌疑
□ 排除大股东高质押
□ 排除行业政策风险

事中风控：
□ 设置止损位（-8%）
□ 设置止盈位（+30%）
□ 仓位上限（20%）

事后风控：
□ 定期复盘（每周）
□ 异动分析（实时）
□ 策略调整（每月）
```
