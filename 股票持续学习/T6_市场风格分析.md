> **版本**：v1.0
> **创建日期**：2026-04-27
> **核心理念**：识天气，知进退 — 每次行动前先看市场风向

---

## 一、什么是市场风格

**定义**：市场风格是指特定时期内，市场资金偏好、情绪状态、运行模式的综合体现。

**为什么重要**：
- 决定什么策略有效（成长 vs 价值）
- 决定什么板块有机会（板块轮动）
- 决定仓位水平（进攻 vs 防守）
- 决定持股时间（短线 vs 中长线）

---

## 二、市场风格分析框架

### 2.1 五维度分析法

| 维度 | 观察指标 | 风格判断 |
|-----|---------|---------|
| **大势** | 上证指数、创业板指走势 | 牛/熊/震荡 |
| **板块** | 领涨/领跌板块、板块轮动 | 主线板块、热点切换 |
| **资金** | 北向资金、两融余额、成交额 | 资金面松紧 |
| **情绪** | 涨停数、跌停数、换手率 | 贪婪/恐惧程度 |
| **估值** | 全市场PE/PB分位 | 高估/低估区域 |

---

### 2.2 大势判断（牛熊震荡）

**牛市特征**：
```
✓ 指数持续上涨，高点不断抬高
✓ 成交量放大，增量资金入场
✓ 涨停家数多（>100家），跌停少（<10家）
✓ 均线多头排列（5>10>20>60）
✓ 新开户数激增，全民谈股
```

**熊市特征**：
```
✗ 指数持续下跌，低点不断降低
✗ 成交量萎缩，存量博弈
✗ 跌停家数多（>50家），涨停少（<30家）
✗ 均线空头排列（5<10<20<60）
✗ 老股民销户，谈股色变
```

**震荡市特征**：
```
○ 指数在一定区间波动（如3000-3300）
○ 成交量中等，存量资金博弈
○ 板块快速轮动，热点持续性差
○ 均线交织缠绕，方向不明
○ 涨跌停均衡（30-50家）
```

---

### 2.3 板块轮动分析

**行业轮动规律**：

| 经济阶段 | 领先板块 | 滞后板块 |
|---------|---------|---------|
| **复苏初期** | 金融、地产、基建 | 防御性板块 |
| **复苏中期** | 周期股、制造业 | 公用事业 |
| **繁荣期** | 消费、科技 | 金融 |
| **衰退初期** | 医药、公用事业 | 周期股 |
| **衰退后期** | 成长股（提前布局） | 消费 |

**观察方法**：
```
每日观察：
- 今日涨幅前5行业
- 今日跌幅前5行业
- 连续3日上涨板块（可能成主线）
- 连续3日下跌板块（可能被弃）
```

---

### 2.4 资金面分析

**北向资金（外资风向标）**：
| 净流入 | 含义 |
|-------|------|
| +50亿以上 | 强势流入，看好A股 |
| +10~50亿 | 温和流入，偏乐观 |
| -10~+10亿 | 平衡，观望 |
| -50亿以下 | 大幅流出，偏谨慎 |

**两融余额（杠杆情绪）**：
```
两融余额上升 → 市场风险偏好提升
两融余额下降 → 市场风险偏好下降
两融余额/流通市值 > 2.5% → 警惕杠杆风险
```

**成交额（市场活跃度）**：
```
上证日成交额：
- >5000亿 → 高度活跃，注意过热
- 3000-5000亿 → 正常活跃
- <3000亿 → 低迷，关注底部信号
```

---

### 2.5 情绪指标

**涨跌停家数**：

| 涨停数 | 跌停数 | 情绪判断 |
|-------|-------|---------|
| >100 | <10 | 极度活跃（贪婪） |
| 50-100 | 10-30 | 活跃 |
| 30-50 | 30-50 | 中性 |
| 10-30 | 50-100 | 萧条 |
| <10 | >100 | 极度恐慌 |

**恐惧贪婪指数**（参考）：
```
使用原则：
极度贪婪（>80）→ 警惕见顶
极度恐惧（<20）→ 关注抄底机会
```

---

### 2.6 估值分位

**全市场估值中枢**：

| 指标 | 分位区间 | 判断 |
|-----|---------|------|
| PE(TTM) | <30% | 低估，可积极布局 |
| PE(TTM) | 30-70% | 合理，正常配置 |
| PE(TTM) | >70% | 高估，谨慎减仓 |

**查询工具**：
- 理杏仁：全市场估值分位
- 乐咕乐股：PE/PB Band
- neodata-financial-search：行业估值

## 三、风格类型与应对

### 3.1 常见市场风格

| 风格类型 | 特征 | 有效策略 | 无效策略 |
|---------|------|---------|---------|
| **成长牛市** | 科技股领涨，估值扩张 | 买成长、追龙头 | 价值投资、防御配置 |
| **价值回归** | 低估值板块补涨 | 买低估、顺周期 | 追高成长股 |
| **防御震荡** | 医药消费强，波动小 | 防御股、高股息 | 追热点、高频交易 |
| **普跌熊市** | 各板块齐跌 | 空仓、减仓 | 抄底、加杠杆 |
| **结构分化** | 部分板块涨，部分跌 | 选对板块、择股 | 指数投资、均衡配置 |

### 3.2 风格切换信号

**从成长转向价值**：
```
信号：
- 科技股滞涨，传统行业启动
- 利率上行，资金偏好变化
- 成长股估值过高，风险释放
```

**从进攻转向防御**：
```
信号：
- 大盘高位放量滞涨
- 领涨板块开始补跌
- 外围市场动荡
- 政策面收紧信号
```

## 六、实战心法

### 6.1 风格识别三原则

1. **顺势而为**：风格确立后跟随，不逆势操作
2. **及时切换**：发现风格变化信号，果断调整策略
3. **保持弹性**：预留仓位，应对风格突变

### 6.2 风格管理清单

```
每周必须做：
□ 更新风格跟踪表
□ 判断当前市场风格
□ 检查持仓与风格匹配度

每月必须做：
□ 回顾风格判断准确性
□ 分析风格切换时机
□ 调整下月风格预判

每次交易前必须问：
□ 当前市场风格是什么？
□ 我的策略与风格匹配吗？
□ 风格可能切换吗？
```
