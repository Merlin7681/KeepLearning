# T3 宏观指标清单

> **版本**：v1.0
> **创建日期**：2026-04-27
> **用途**：宏观分析核心工具，理解经济周期与政策对A股的影响

---

## 一、核心经济指标

### 1. GDP（国内生产总值）

**发布频率**：每季度（初值+修正值）
**发布时间**：季度结束后15-20天

**含义**：经济总量的晴雨表

**判断标准**：
| GDP增速 | 经济状态 | A股影响 |
|--------|---------|---------|
| >6% | 高增长 | 利好周期股 |
| 5-6% | 稳健增长 | 正常环境 |
| 4-5% | 增速放缓 | 政策托底预期 |
| <4% | 低增长 | 防御板块避险 |

**A股传导路径**：
```
GDP ↑ → 企业盈利↑ → 顺周期板块（银行、券商、周期）表现好
GDP ↓ → 政策刺激预期↑ → 成长板块（新能源、半导体）博弈机会
```

---

### 2. CPI（消费者物价指数）

**发布频率**：每月
**发布时间**：次月9-10日

**含义**：通胀水平，影响货币政策方向

**判断标准**：
| CPI同比 | 通胀状态 | 货币政策倾向 |
|--------|---------|-------------|
| >3% | 通胀压力 | 收紧倾向 |
| 2-3% | 温和通胀 | 中性 |
| 0-2% | 低通胀 | 宽松空间 |
| <0% | 通缩风险 | 强刺激预期 |

**A股传导路径**：
```
CPI高企 → 央行收紧 → 利空成长股（估值承压）
CPI低迷 → 宽松预期 → 利好成长股（估值扩张）
CPI温和 → 货币中性 → 业绩驱动为主
```

**重点关注**：
- 🔍 猪肉价格（CPI权重约2.5%）
- 🔍 服务价格（反映消费复苏）

---

### 3. PPI（生产者物价指数）

**发布频率**：每月
**发布时间**：次月9-10日

**含义**：工业企业出厂价格，反映周期盈利

**判断标准**：
| PPI同比 | 含义 | 受益板块 |
|--------|------|---------|
| >5% | 上游涨价强 | 煤炭、有色、化工 |
| 0-5% | 温和上涨 | 制造业利润承压 |
| <0% | 工业通缩 | 上游盈利恶化 |

**A股传导路径**：
```
PPI上行 → 上游企业利润改善 → 资源股行情
PPI下行 → 中游制造业成本下降 → 消费电子、家电受益
PPI-CPI剪刀差 → 利润在上中下游分配
```

---

### 4. PMI（采购经理人指数）

**发布频率**：每月
**发布时间**：当月最后一天或次月1日

**含义**：经济先行指标，反映企业景气度

**判断标准**：
| PMI值 | 含义 | 市场解读 |
|-------|------|---------|
| >53 | 扩张强劲 | 利好顺周期 |
| 50-53 | 温和扩张 | 正常 |
| 48-50 | 收缩边缘 | 政策托底预期 |
| <48 | 明显收缩 | 避险情绪升温 |

**重点关注**：
- 🔍 制造业PMI（权重最大）
- 🔍 新订单指数（需求前瞻）
- 🔍 出口订单指数（外需变化）

**A股传导路径**：
```
PMI连续站上50 → 经济复苏确认 → 顺周期板块补涨
PMI跌破50 → 经济放缓 → 政策刺激预期 → 成长股反弹
PMI与股指背离 → 跟随修复（通常PMI领先1-2个月）
```

---

### 5. 社会融资规模（社融）

**发布频率**：每月
**发布时间**：次月10-15日

**含义**：实体融资需求，反映信用周期

**判断标准**：
| 社融增速 | 含义 | A股影响 |
|---------|------|---------|
| >12% | 宽信用强 | 利好成长股（流动性充裕） |
| 10-12% | 温和扩张 | 震荡市 |
| <10% | 信用收缩 | 利空成长股（估值承压） |

**重点关注**：
- 🔍 新增人民币贷款（占比约70%）
- 🔍 企业中长期贷款（投资意愿）
- 🔍 居民贷款（消费+购房）

**A股传导路径**：
```
社融大幅新增 → 宽信用确认 → 市场底部区域（历史规律）
社融持续收缩 → 信用紧缩 → 成长股估值杀跌
社融-M2剪刀差 → 实体融资需求强度
```

## 二、货币市场指标

### 1. LPR（贷款市场报价利率）

**发布频率**：每月20日

**含义**：实体融资成本基准

**判断标准**：
| LPR变化 | 货币政策信号 | A股影响 |
|--------|-------------|---------|
| 下调5BP+ | 明确宽松 | 利好成长股 |
| 下调5BP以内 | 温和宽松 | 中性偏多 |
| 持平 | 政策观察 | 中性 |
| 上调 | 收紧信号 | 利空 |

**A股传导路径**：
```
LPR下调 → 融资成本下降 → 利好高负债行业（地产、基建）
LPR下调 → 估值分母下降 → 利好成长股（DCF估值改善）
```

---

### 2. MLF（中期借贷便利）

**发布频率**：每月15日（操作日）

**含义**：央行投放中期基础货币的工具

**重点关注**：
- 🔍 MLF利率（政策利率）
- 🔍 MLF投放量（对冲到期情况）

**A股传导路径**：
```
MLF利率下调 → LPR下调预期 → 利好股市
MLF净投放 → 流动性宽松 → 利好债市和成长股
MLF净回笼 → 流动性收紧 → 利空
```

---

### 3. 存款准备金率（RRR）

**含义**：银行须缴存的准备金比例

**判断标准**：
| RRR调整 | 含义 | A股影响 |
|--------|------|---------|
| 下调50BP+ | 大幅宽松 | 强利好（历史平均涨幅5%） |
| 下调25BP | 温和宽松 | 利好 |
| 持平 | 政策观察 | 中性 |
| 上调 | 收紧 | 利空 |

---

### 4. M2（广义货币供应量）

**发布频率**：每月
**发布时间**：次月10-15日

**含义**：市场流动性总闸门

**判断标准**：
| M2增速 | 含义 | A股影响 |
|--------|------|---------|
| >12% | 流动性充裕 | 利好成长股 |
| 8-12% | 正常水平 | 中性 |
| <8% | 流动性偏紧 | 利空 |

**M2-GDP剪刀差**：
```
M2增速 > GDP增速 + CPI → 流动性过剩 → 资产价格上涨
M2增速 < GDP增速 + CPI → 流动性紧张 → 资产价格承压
```

## 三、利率与汇率

### 1. 10年期国债收益率

**含义**：无风险利率锚，影响股市估值

**判断标准**：
| 收益率水平 | 含义 | A股影响 |
|-----------|------|---------|
| <2.5% | 利率低位 | 利好成长股估值 |
| 2.5-3.5% | 正常区间 | 中性 |
| >3.5% | 利率高位 | 利空成长股估值 |

**A股传导路径**：
```
利率下行 → DCF分母下降 → 成长股估值扩张
利率上行 → 估值承压 → 高股息板块防御价值提升
利率与股息率差 → 股市性价比判断
```

---

### 2. 美联储利率（Fed Funds Rate）

**含义**：全球资产定价锚，影响人民币汇率和资金流向

**判断标准**：
| Fed利率趋势 | 含义 | A股影响 |
|------------|------|---------|
| 加息周期 | 美元回流 | 利空新兴市场 |
| 降息周期 | 美元外溢 | 利好新兴市场 |
| 维持高位 | 汇率压力 | A股估值承压 |

**A股传导路径**：
```
美联储加息 → 人民币贬值压力 → 外资流出 → A股承压
美联储加息 → 中美利差倒挂 → 货币政策受限
美联储降息预期 → 美元走弱 → 人民币企稳 → 利好A股
```

---

### 3. 人民币汇率（USD/CNY）

**含义**：影响外资流入意愿和出口竞争力

**判断标准**：
| 汇率趋势 | 含义 | A股影响 |
|--------|------|---------|
| 升值（破7） | 资金流入 | 利好A股 |
| 稳定（7-7.3） | 正常波动 | 中性 |
| 贬值（>7.3） | 资金流出压力 | 利空A股 |

**A股传导路径**：
```
人民币升值 → 外资流入增加 → 核心资产（茅台、宁德）受益
人民币贬值 → 出口型企业利润改善 → 纺织、电子代工受益
汇率大幅波动 → 央行干预预期 → 市场情绪敏感
```

---

## 四、国际大宗商品

### 1. 原油（WTI/Brent）

**含义**：通胀源头，影响化工、航空、物流等板块

**判断标准**：
| 油价水平 | 含义 | A股影响 |
|--------|------|---------|
| >100美元 | 高油价 | 利空航空、物流；利好油气开采 |
| 70-100美元 | 中高位 | 正常水平 |
| <70美元 | 低油价 | 利好航空、炼化；利空油气 |

**A股传导路径**：
```
油价上涨 → 油气开采（中国石油、中海油服）受益
油价上涨 → 化工品涨价（万华化学、恒力石化）利润改善
油价上涨 → 航空、物流成本上升，利空
油价上涨 → 输入性通胀预期 → 货币政策收紧压力
```

---

### 2. 铜（LME铜）

**含义**：经济周期晴雨表，"铜博士"

**判断标准**：
| 铜价趋势 | 含义 | A股影响 |
|--------|------|---------|
| 上涨 | 经济复苏 | 利好有色、制造业 |
| 震荡 | 经济平稳 | 中性 |
| 下跌 | 经济放缓 | 利空周期股 |

**A股传导路径**：
```
铜价上涨 → 紫金矿业、江西铜业受益
铜价上涨 → 制造业成本上升，利润承压
铜价与PMI共振 → 确认经济趋势
```

---

### 3. 黄金（COMEX黄金）

**含义**：避险资产，反映风险偏好

**判断标准**：
| 黄金趋势 | 含义 | A股影响 |
|--------|------|---------|
| 上涨 | 避险情绪升温 | 利好黄金股（山东黄金、紫金矿业） |
| 下跌 | 风险偏好提升 | 利好成长股 |

**A股传导路径**：
```
黄金大涨 → 避险情绪强 → A股承压
黄金大涨 → 黄金股独立行情
```

---

### 4. 锂（碳酸锂价格）

**含义**：新能源产业链景气度

**判断标准**：
| 锂价水平 | 含义 | A股影响 |
|--------|------|---------|
| >20万/吨 | 高景气 | 利好锂矿股 |
| 10-20万/吨 | 正常水平 | 中性 |
| <10万/吨 | 供过于求 | 利空锂矿股 |

**A股传导路径**：
```
锂价上涨 → 天齐锂业、赣锋锂业业绩改善
锂价下跌 → 新能源车成本下降，利好整车厂（比亚迪）
锂价趋势 → 影响整个新能源板块估值
```

## 六、宏观周期判断框架

### 经济周期四阶段

```
        GDP↑
          │
    复苏期│过热期
          │
    ──────┼────── CPI
          │
    衰退期│滞胀期
          │
        GDP↓
```

| 周期阶段 | 特征 | 表现强势板块 |
|---------|------|-------------|
| **复苏期** | GDP↑ CPI↓ | 金融、可选消费 |
| **过热期** | GDP↑ CPI↑ | 周期、商品 |
| **滞胀期** | GDP↓ CPI↑ | 现金、黄金、能源 |
| **衰退期** | GDP↓ CPI↓ | 公用事业、必选消费 |
